How Minimum-of-K Forces Diversity

Step-by-step breakdown of the winner-takes-all loss logic

01
📐

Compare All Modes

The ground truth curve is compared against all K=3 predicted trajectories using Huber Loss (Smooth L₁).

loss_k = SmoothL1(pred_k, gt)
02
🏆

Find the Winner

For each sample, find the mode index with the minimum loss — this is the "best guess".

best = argmin(losses, dim=1)
03
⚡

Reward the Winner

Only backpropagate gradients through the winning mode. The other two modes receive zero penalty.

loss = losses[best_mode]
04
🌊

Modes Diversify

Without punishment, losing modes are free to explore different directions, naturally covering the multi-modal future space.

diversity → guaranteed

The Mode Collapse Problem

Why standard MSE destroys multi-modality

❌ Standard MSE

Mode Collapse
